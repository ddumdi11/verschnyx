# Sau-durchs-Dorf-treiben|omx

Naturgemäß spielen hier die Medien eine Schlüsselrolle. Es herrscht aber mittlerweile hierzulande (so kommt es mir jedenfalls vor) eine gewisse Ausgewogenheit, so dass zwar zum Treiben angesetzt werden kann, es aber meist gleich darauf zum Abkühlen unter die Dusche geht, oder dass eine Seite antreibt, eine andere zügelt.
