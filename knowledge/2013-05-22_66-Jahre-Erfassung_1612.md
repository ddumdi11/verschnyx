Die Punkte sind zum Beispiel Ausdrücke, für die es abweichende Definitionen

Oft ist es auch gar nicht gewollt (besonders in politischen Diskussionen),

richtig sind.  
  
Aber das größte Problem ist die Zeit.  
Man kann vom Hundersten ins Tausendste kommen.  
  
Beispiel "Urteile/Bewertungen" oben:  
  
- "gleicher Wissensstand" bedeutet dieselben Informationen  
- "ähnliche Schlussfolgerungen" bedeutet ähnliche methodische Fertigkeiten  
  
"Dieselben Informationen":

Erkennt man eine Statistik an?  
Usw. ...  
  
  
"Ähnliche methodische Fertigkeiten":  
  
Vermutlich werden in den seltensten Fällen echte Methoden angewandt.  
  
Wahrscheinlich ist es aber meistens ein einfaches "von-oben-nach-unten"-Muster:  
Alle vorhandenen Informationen werden als Elemente von Ebenen sortiert.  
Von der niedrigsten Ebene wird aus Elementen eine Schlußfolgerung gezogen,  
die auf einer der nächsthöheren Ebenen als neues Element zu den vorhandenen  
kommt.  
Daraus folgt dann die nächste Schlußfolgerung, bis man oben ist.  
Vielleicht ergibt sich dem Informationsgehalt einer Schlußfolgerung  
eine Umordnung von Elementen, aus denen man weiter schließen will.  
Theoretisch kann das Ganze ziemlich kompliziert werden, weil es  
viele Schlußfolgerungen von einer Ebene zur nächsten geben kann  
und weil man auf jeder Ebene ein Element mehrfach in vielen  
Zusammenstellungen für neue Schlußfolgerungen mit anderen Elementen  
verwenden kann.  
Das Ergebnis, worauf man hinaus will, muss gar nicht ganz oben sein.  
"Ergebnis" kann auch bedeuten: drei Schlußfolgerungen, jede auf einer  
anderen Ebene.  
Denkbar und machbar ist vieles.
