# Je mehr Bilderkennungsarbeit, desto mehr Bilder

Je mehr (hinein) interpretiert wird, desto mehr ...
Je mehr Bilderkennungsarbeit geleistet wird, desto mehr Bilder entstehen.
Wieviele Bilder entstehen?
Stimmt diese Behauptung?
Anzahl entstehender Bilder = Anzahl richtig erkannter Bilder + Anzahl falsch erkannter Bilder
Das Verhältnis, die Relation von richtig und falsch erkannten Bildern zueinander
Anzahl richtig erkannter Bilder : Anzahl falsch erkannter Bilder
z.B.:
3 : 1
ist abhängig von der Sehleistung, die widerum abhängt von der Schärfe erkannter Bilder - welche wiederum abhängig ist von der Brille, die ein/eine Bilderkenner/Bilderkennerin trägt, sofern die Augen alleine nicht völlig klare/scharfe Bilder (mehr) sehen können.
