# Induktiver Gewaltakt

Sender-Empfänger-Modell
Defintition:
Ein Sender muß Energie einsetzen, damit es zu einer Sendung von Gewalt kommt.
Ein Empfänger muß Energie einsetzen, damit es zum Vollzug des Gewaltakts kommt.
Da gibt es - glaube ich - drei Möglichkeiten. Bei 1 und 3 wird der Gewaltakt vollzogen, aber nur bei 1 eigentlich als vollgültig (oder wie fim es nennen mag).
1
Ein Sender sendet, setzt Energie ein.
Ein Empfänger empfängt, setzt Energie ein.
Der Gewaltakt wird durch den Empfänger ausgeführt.
2
Ein Sender sendet, setzt Energie ein.
Ein Empfänger empfängt, setzt aber keine Energie ein.
Der Gewaltakt wird durch den Empfänger nicht ausgeführt.
3
Ein Sender sendet nicht, setzt keine Energie ein.
Ein Empfänger empfängt nichts, setzt aber Energie ein.
Der Gewaltakt wird durch den Empfänger ausgeführt.
Im Fall 3 kann (wissentlich fälschlicherweise) behauptet werden, es hätte eine Sendung stattgefunden. Das dürfte die bekannteste Variante sein, eine von mehreren.
