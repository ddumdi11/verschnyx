# "Die Kugel flog so nah an mir vorbei, dass ich den Luftzug spürte."

></a>"Die Kugel flog so nah an mir vorbei, dass ich den Luftzug spürte."</h2><p>Die intensivste Kampferfahrung habe das Ex-Model aber erst danach gemacht. Ein deutscher Kämpfer habe ihr gesagt, sie würden demnächst direkt an der Front mitkämpfen. Innerhalb der Einheit, bestehend aus sechs westlichen Kämpfern, war Bohman die einzige Frau. Der Kampf gegen den IS habe nahe der <a href="http://www.focus.de/orte/tuerkei/">türkischen</a> Grenze stattgefunden. "Das war das letzte Mal, dass ein Heckenschütze auf mich schoss. Die Kugel flog so nah an mir vorbei, dass ich den Luftzug spürte."</p><p>Nachdem Bohman fast 15 Kilo verlor, kehrte das Ex-Model zurück nach Kanada. Sie konnte jedoch nicht von dem Gedanken ablassen, in Rojava für ihre Ideale zu kämpfen. Bereits im September kehrte sie zurück an die Front. Bis heute ist sie da geblieben.</p><h2>
