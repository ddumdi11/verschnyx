# Querulanzien-Nährböden - billig erhältlich

></a>Querulanzien-Nährböden - billig erhältlich</h3><p>Als Querulant (von <a href="https://de.wikipedia.org/wiki/Latein">lateinisch</a> queri - „vor Gericht klagen") wurden ursprünglich in der <a href="https://de.wikipedia.org/wiki/Judikative">Rechtsprechung</a> Menschen bezeichnet, die trotz geringer Erfolgsaussicht besonders unbeirrbar und zäh einen Rechtskampf führen. Dabei steht ein geringfügiger oder vermeintlicher Anlass kaum noch im Verhältnis zum rechthaberischen, misstrauischen, fanatischen und unbelehrbaren Vorgehen der so bezeichneten Menschen. Auch Personen, die bei <a href="https://de.wikipedia.org/wiki/Beh%C3%B6rde">Behörden</a> oder vor <a href="https://de.wikipedia.org/wiki/Gericht">Gericht</a> ständig unbegründete <a href="https://de.wikipedia.org/wiki/Antrag">Anträge</a> stellen, werden als Querulanten bezeichnet. Später wurde der Begriff von der <a href="https://de.wikipedia.org/wiki/Psychiatrie">Psychiatrie</a> aufgegriffen und entweder als eigenes, <a href="https://de.wikipedia.org/wiki/Wahn">wahnhaftes</a>, oft <a href="https://de.wikipedia.org/wiki/Paranoia">paranoides</a> Krankheitsbild, <a href="https://de.wikipedia.org/wiki/Pers%C3%B6nlichkeitsst%C3%B6rung">Persönlichkeitsstörung</a> oder als begleitendes Symptom anderer <a href="https://de.wikipedia.org/wiki/Psychische_St%C3%B6rung">psychischer Störungen</a> beschrieben. Die Anwendung des Begriffs ist rechtlich problematisch, steht doch beim Vorliegen einer derartigen Störung die <a href="https://de.wikipedia.org/wiki/Prozessf%C3%A4higkeit_%28Recht%29">Prozessfähigkeit</a> in Frage. Dadurch kann ein Betroffener effektiv an der missbräuchlichen Nutzung von Klagen und <a href="https://de.wikipedia.org/wiki/Rechtsbehelf">Rechtsbehelfen</a>, aber auch an der Durchsetzung tatsächlicher Rechte gehindert werden.
