# "Kein", "nicht" - misten Sie Ihren Wortschatz aus!

></a>"Kein", "nicht" - misten Sie Ihren Wortschatz aus!</h2><p>Der Negativdenker gibt sich oft erst auf den zweiten Blick zu erkennen. Im ersten Moment klingen seine Aussagen positiv. Jedoch sind seine häufigsten Wörter „nicht" und „kein". Aus diesem Grund erzeugen seine Aussagen nämlich die gegenteilige Wirkung.</p><p>Woran liegt das? Das menschliche <a href="http://t.ktxtr.com/redir?pid=23&cid=60&format=rlink&kw=Gehirn&sid=90cabe66-2fc1-42ef-9788-3b1d87850213&target=http%3A%2F%2Fapp.kontextr.eu%2Fcontent%2FamazonRedirect%3Fp%3D23%26u%3Dhttp%3A%2F%2Fwww.amazon.de%2Fs%2F%253Ffield-keywords%3DGehirn">Gehirn</a> kann diese Worte „nicht" und „kein" nicht verarbeiten. Stellen Sie sich mal keine saure Zitrone vor. Vermutlich taucht gerade jetzt das Bild einer gelben sauren Zitrone vor Ihrem geistigen Auge auf. Oder denken Sie nicht an einen frisch gebrühten <a href="http://www.focus.de/gesundheit/video/wundermittel-oder-gesundheitskiller-die-wahrheit-ueber-kaffee_vid_39461.html?ucil=Video_Kaffee">Kaffee</a>. Wahrscheinlich können Sie ihn schon riechen.</p><p>Aus: <a href="http://www.focus.de/finanzen/experten/schmidt/positiv-formulieren-ich-will-nicht-ich-kann-nicht-so-kommen-sie-aus-der-negativ-spirale_id_5079415.html">http://www.focus.de/finanzen/experten/schmidt/positiv-formulieren-ich-will-nicht-ich-kann-nicht-so-kommen-sie-aus-der-negativ-spirale_id_5079415.html</a></p><p><br></p><p>Dank diesem oben verlinkten und zitierten Artikel</p><p>rückt ein weiterer (mal wieder vergessener) Aspekt</p><p>wieder an seine richtige Stelle in mein Bewusstsein:<br></p><h2>
