# Das Virtuell-Bild|OMX

Das ist das Bild, was auf seiner Ebene versucht, ein Bild von anderer Ebene (einer höheren, logischerweise) zu vertreten, an dessen STelle zu stehen - weil dieses Bild, was so vertreten ist und wird, selber nicht stehen kann, da wo man/fim es verlangen iwi grad so kann, ohne mit Zuchthaus-PKH bestraft werden zu müssen.
