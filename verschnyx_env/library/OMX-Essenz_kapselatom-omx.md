# Kapselatom|omx

Das kleinste unteilbare Element innerhalb einer Kapselung (gemäß des Sinnes der ursprünglichen Benennung der sog. Alten Griechen). Es kann nur eines geben! Sobald von einer Sorte mehrere verschiedene "auftauchen", darf fim sich in Gewissheit wiegen, dass es sich doch nicht um das Atom gehandelt hat, was das kleinste unteilbare Kapselelement darstellt: Es muss sich dann wohl um eine Gruppe anderer Schauspieler handeln - ganz klar: die fim dann auch anders benennen sich tut.
