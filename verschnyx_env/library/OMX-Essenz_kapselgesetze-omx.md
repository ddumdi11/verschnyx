# Kapselgesetze|omx

Es gibt Gesetzmäßigkeiten, die innerhalb einer Kapselung (z.B. Natur ;-)) auftreten und zu Änderungen innerhalb des Kapselsystems führen, aber die Kapselung selbst, das System selbst nicht zu ändern vermögen.
